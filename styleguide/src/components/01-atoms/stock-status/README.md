# Stock status v1.0.0

Displays an articles' stock status in a span element. Three modifiers are available: in-stock, few-in-stock and out-of-stock.

## Browser compatibility

* Chrome 66 (also on Android)
* Safari 11 (also on iOS)
* IE11
