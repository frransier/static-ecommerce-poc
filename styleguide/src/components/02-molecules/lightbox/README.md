# lightbox v1.0.0

A simple lightbox.

## Author(s)

- Sebastian Gelotte
- Mikael Karlsson

## Dependencies

- `check-if-ie` SCSS mixin
- `fixed-cover-viewport` SCSS mixin
- `arrow-right` SVG
- `arrow-left` SVG

## Browser compatibility

Tested in the following browsers:

- Chrome 72
- Firefox 65
- Edge
- IE11
- Safari 11 on iOS
