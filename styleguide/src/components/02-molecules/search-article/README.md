# Search article v1.1.0

Displays basic information about an article, for use in the Search component.

## Author(s)

Mikael Karlsson

## Dependencies

- `mq` mixin

## Browser compatibility

Tested in the following browsers:

- Firefox 61
- Chrome 66 (also on Android)
- Safari 11 (also on iOS)
- IE 11
