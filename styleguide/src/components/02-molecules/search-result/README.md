# Search result v1.0.0

A list displaying a search result (articles) with some minor styling applied to it.

## Author(s)

Mikael Karlsson

## Dependencies

- `list-clean` mixin
- `mq` mixin

## Browser compatibility

Tested in the following browsers:

- Firefox 61
- Chrome 66 (also on Android)
- Safari 11 (also on iOS)
- IE 11
